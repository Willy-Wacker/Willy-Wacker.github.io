Neue Website

Willy Wacker, 2020-06-27

Ich sterbe an der Hitze in Köln und ausgerechnet heute mache ich diesen Blog auf. Klarer Fall von Irrsin! Aber ich wollte mal sehen, wie man eine Website bei Github anlegt. Ist gar nicht so schwer, sobald man weiß, wie es geht. Man muss nur ein Repro anlegen, welches den Namen username.github.io hat. Darin wird einfach die Website abgelegt, für den Anfang meist einfach ein index.html. Man legt alles auf dem eigenen Rechner an und schickt es auf dem üblichen Weg zu Github.
